# rsonar: R Code Quality Analysis Inspired by SonarQube

Provides a unified interface for R code quality analysis inspired by
SonarQube. Combines lintr (static analysis), styler (code style), covr
(test coverage) and goodpractice (best practices) into comprehensive
quality reports. Estimates technical debt, detects code smells, and
integrates seamlessly into CI/CD pipelines (GitLab CI, GitHub Actions).
Produces HTML, JSON and JUnit XML reports suitable for SonarQube
ingestion.
